# Premium Portfolio - Rafiul Kabir

A fully responsive, modern, AI and cybersecurity-inspired portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

## Features
- **Dark Theme UI**: Glassmorphism, neon glow effects, floating particles.
- **Responsive**: Mobile, tablet, desktop, and ultrawide support.
- **Animations**: Smooth scroll reveal and hover effects using Framer Motion.
- **Components**: Hero, About, Skills, Education, Projects, Certifications, Resume, Contact.

## Prerequisites
To run this project locally, you must install **Node.js** from [nodejs.org](https://nodejs.org).

## Installation
1. Open a terminal in this directory.
2. Install dependencies:
   ```bash
   npm install
   ```

## Running Locally
Start the development server:
```bash
npm run dev
```

## Building for Production
```bash
npm run build
```
This will create a `dist` directory which you can upload to any static hosting provider (Vercel, Netlify, GitHub Pages, Firebase Hosting).
